# march-madness-ml-model
End-to-end March Madness prediction model built with Python, leveraging historical NCAA data, feature engineering (seed, win percentage, point differential), and logistic regression to generate probabilistic game outcomes and Kaggle submissions.


Abstract:
This project develops a machine learning model to predict outcomes of NCAA March Madness games using historical team performance data. By leveraging feature engineering and logistic regression, the model generates probabilistic predictions for tournament matchups. The approach emphasizes interpretability, statistical rigor, and practical application in bracket prediction and Kaggle-style competitions.

Methodology:
Engineered matchup-based features: seed difference, win percentage difference, and point
differential difference.
Trained a logistic regression model for probabilistic predictions.
Evaluated performance using log loss to ensure calibrated outputs.

Results:
Achieved ~0.51 validation log loss.
Identified key predictive features influencing game outcomes.
Generated submission-ready predictions for tournament matchups.

Skills & Tools:
Python, Pandas, NumPy, Scikit-learn, Data Cleaning, Feature Engineering, Model Evaluation

Impact:
Demonstrates ability to build end-to-end predictive models, apply statistical reasoning, and translate
data into actionable skills directly applicable to quantitative finance and data science roles.
